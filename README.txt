Для запуска проекта (на данный момент) необходимо сделать следующее:

1) Установить Pycharm + Anaconda, открыть проект в данной IDE и импортировать необходимые библиотеки
2) Запустить анализатор Svace на коде juliet
3) Переписать ProjectConfig (СМ комментарии)
- xml_source_path
- code_source_path
4) зайти в build и сделать:
cmake ../
5) можно запускать тесты из папки tests. Первым стоит запустить main_test
